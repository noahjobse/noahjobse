### SAIT-ARIS Robot Simulator *(Private — SAIT Research Partnership)*
**React 19 · Babylon.js 8 · FastAPI · PyBullet · Lark · Docker**

Web-based ABB industrial robot trajectory simulator built in partnership with SAIT Research Applications × ARIS Robotics.
- Renders real-time 3D robot motion via **Babylon.js WebGL/WebGPU** from RAPID program uploads
- Built a **grammar-based RAPID language parser** (Lark) → AST → trajectory pipeline fed into a PyBullet IK solver
- Implemented **mesh-to-mesh collision detection** and physics-accurate inverse kinematics for ABB IRB 4600/2600/6700 robots
- Containerized full stack (React + Vite / FastAPI + Python) with Docker Compose
